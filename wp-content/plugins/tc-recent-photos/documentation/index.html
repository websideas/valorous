How to Install

Installing this plugin is super easy and only takes a few seconds!

1. In your WordPress dashboard add a new plugin.

2. Select the tc-recent-photos.zip file for upload, NOT the file you downloaded from CodeCanyon. If you select the wrong file you will get an error and the plugin will not install.

3. Activate the plugin and go to the settings page for Instagram Widget Pro.

4. Next you will have to setup your site to access the Instagram API. You will see under your Client ID in the settings page there is a redirect URI. You will need to either change your app, or create a new app using this URL for the redirect. If you do not set this up properly you will not be able to generate tokens to show images. Once your App is setup enter your Instagram "client ID" in the settings. Click save and then click Generate Token in the Access Token settings. Follow the prompts in the box.

Using the Widget

Under the Apperance tab of your Dashboard click widgets:

You will now have a new widget listed that you can drag into your sidebar areas. Drag the widget over and it will expand to display the options and their default values. Configure the widget as desired and click save! Your widget will then be live in the side bar.

Widget Title - Enter the title of the widget area, you can leave this blank to omit the heading tag and title text.

Show Images From - In this drop down you can select to show images from a user (User ID), or look for images using a tag.

Instagram Username - Enter the user ID to get images from. You can get the ID of a username here: http://jelled.com/instagram/lookup-user-id 

Instagram Tag - Enter a valid instagram tag to get recent photos from. Example: "blonde girls" (without quotes!)

Image Count - Enter the number of images to display (12 max)

Image Size - Enter the size of thumbnail to use.

Columns - Enter the number of columns to use, this allows you to create a grid to fit your needs by adjusting this value with the image size and count.

Lightbox - Enable a lightbox that shows the full size images when clicked, with gallery support.

Follow Button - You can check this to enable a follow button back to your Instagram profile.

Using the Shortcode

To use the widget in pages, posts, etc. you can use the tinyMCE plugin to generate a shortcode for you.

Simply click the instagram camera icon in your tinyMCE toolbar (visual mode) and follow the settings just like you did for the widget setup. When you have your settings laid out click insert shortcode and everything will be inserted into the post for you.

Load Times / Caching

Note that all calls to instagram and the images themselves are cached within WordPress. That means that once a users feed has been called and the images cached, that data will load from the cache instead of fetching everything on every page load. You may notice a little longer loading time on first run (as the cache is being created) but after that the data is coming all from local so it will run as fast as if you uploaded the images to your site and displayed them in a widget.

The cache re-new period is 3 hours.
